O que é Desenvolvimento de Softwares?

Desenvolvimento de software é o processo de conceber, projetar, programar, testar e manter programas de computador e sistemas de software. É uma disciplina ampla que engloba todas as atividades envolvidas na criação e evolução de software, desde a concepção da ideia até a entrega do produto final e sua subsequente manutenção

Quais os principais aspectos envolvidos no desenvolvimento de software?

São: Requisitos, Design, Implementação, Testes, Implantação, Manuteção, Gestão de Projeto e a Documentação

Quais são as principais linguagens que o desenvolvedor de software pode utilizar?

JavaScript; Python; Java; C#; PHP; Ruby; C++, entre outras

Quais foram as linguagens utilizadas para construir o site TecnoFy?

Nós toda equipe do Tecnofy, como principal utilizamos o JavaScript mas usamos muito as linguagens de demarcação de HyperText HTML E CSS.
